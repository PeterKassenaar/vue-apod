# vue-apod

A sample Vue project using the Astronomy Picture of the Day (APOD) API by NASA.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
